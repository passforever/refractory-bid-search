# 🔥 耐火材料招标采购信息搜索工具

> 一键运行，自动全网搜索耐火材料招标采购、窑炉维修信息

## 📋 功能特点

- **多源搜索**：百度、必应、招标网站等多个数据源
- **智能分类**：自动将结果分为「招标采购」「材料采购」「窑炉维修」三大类
- **相关度评分**：根据关键词匹配度、时效性等自动评分排序
- **自动去重**：相同信息自动过滤，避免重复
- **精美报告**：生成可视化HTML报告，浏览器直接查看
- **一键运行**：双击批处理文件即可运行

## 🚀 快速开始

### 1. 环境要求
- Python 3.8+（需已安装并加入 PATH）
- 网络连接

### 2. 安装依赖（首次运行）
```bash
pip install -r requirements.txt
```

### 3. 运行
**方式一（推荐）**：双击 `运行搜索.bat`

**方式二**：命令行运行
```bash
python main.py
```

## 🔍 搜索关键词

工具内置了以下关键词组合：

### 招标/采购类
- 耐火材料 招标/采购/竞标/比价/竞争性谈判

### 窑炉类
- 高炉/热风炉/焦炉/窑炉/加热炉/回转炉 维修招标

### 材料类
- 高铝砖/粘土砖/轻质砖/硅砖/不定型材料/浇注料/散料/泥浆 招标采购

## ⚙️ 自定义配置

编辑 `config.py` 可修改：
- 搜索关键词（`BID_KEYWORDS` / `FURNACE_KEYWORDS` / `MATERIAL_KEYWORDS`）
- 启用/禁用搜索引擎（`SEARCH_SOURCES`）
- 日期过滤范围（`MAX_DAYS_OLD`）
- 排除关键词（`EXCLUDE_KEYWORDS`）
- 请求间隔（`REQUEST_DELAY_MIN/MAX`）

## 📁 项目结构

```
refractory-bid-search/
├── main.py              # 主程序入口
├── config.py            # 配置文件（关键词、搜索源等）
├── search_engine.py     # 核心搜索引擎
├── report_generator.py  # HTML报告生成器
├── requirements.txt     # Python依赖
├── 运行搜索.bat          # 一键启动脚本
└── reports/             # 报告输出目录
    └── 耐火材料招标_2026-06-06.html
```

## 📊 报告示例

运行后自动在 `reports/` 目录生成HTML报告，包含：
- 统计概览（总数、分类数量）
- Tab切换查看不同分类
- 每条结果包含：标题、链接、摘要、来源、关键词、日期、相关度评分
- 点击标题可直接跳转原始页面

## ⚠️ 注意事项

1. 搜索过程需要网络连接
2. 百度等搜索引擎有反爬机制，工具已内置随机延迟，但仍可能偶尔被限制
3. 如遇到验证码，可在浏览器中手动完成验证后重试
4. 搜索结果仅供参考，具体信息请以原始网站为准
5. 建议每天运行一次，获取最新招标信息
